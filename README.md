# Calculator App

## Project Description

This project is a responsive calculator web application created for the EESE316 Software Engineering 3 FA3 assessment. The calculator allows the user to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

The design is based on the Frontend Mentor calculator app challenge. The calculator includes a display screen, digit buttons, operation buttons, a delete button, a reset button, and an equal button.

## Features

- Responsive calculator layout
- Display screen for entered values and results
- Digit buttons from 0 to 9
- Arithmetic operation buttons for addition, subtraction, multiplication, and division
- Decimal point support
- Delete button
- Reset button
- Equal button to calculate the answer
- Clean purple-themed user interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Visual Studio Code
- Web browser for testing

## Development Process

I started by creating the project folder and organising the files into separate folders for CSS and JavaScript. I then created the HTML structure for the calculator, including the display screen and all required buttons.

After completing the HTML, I used CSS to style the calculator and make it look clean, user-friendly, and responsive on desktop and mobile screens. I used CSS Grid to arrange the buttons neatly.

Finally, I wrote JavaScript code to handle the calculator functionality. The JavaScript allows the user to enter numbers, choose arithmetic operations, delete values, reset the calculator, and display accurate results.

## Challenges Faced

One challenge was making sure that the calculator did not allow invalid inputs such as repeated decimal points or operators placed incorrectly. I solved this by adding checks in the JavaScript code before updating the display or calculating the result.

Another challenge was making the calculator responsive on smaller screens. I solved this by using media queries in the CSS file to adjust the button sizes, display size, padding, and spacing.
